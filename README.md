## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

To see it running in development mode with webpack noting changes and using nodemon:

```sh
npm run build:dev
npm run start:dev
```

For production mode:

```sh
npm run build:prod
npm start
```
